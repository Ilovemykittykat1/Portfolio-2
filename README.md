Katherine McNeil Portfolio and Resume Website

This website is my personal portfolio and resume space, built to showcase my work, skills, and creative projects as an IT Programming student and web developer. It blends solid technical structure with visual storytelling through clean layouts, subtle animations, and consistent branding.

Purpose

The goal of this site is to present a clear and engaging overview of my background, including:

• Education in IT Programming
• Certifications in IT, tax, and generative AI
• Technical and programming skills
• Work experience in IT support and website consulting
• Creative projects including my self-published children’s books and custom website builds

Overview

This portfolio brings together my programming experience, digital design work, and writing projects into one cohesive personal brand. The Resume and About sections highlight my qualifications using a clean, professional layout with animated accents and responsive design.

Technologies Used

• HTML5 for semantic structure
• CSS3 for layout, styling, and responsiveness
• JavaScript for interactivity and canvas animation
• Animate.css for entrance animations
• AOS Animate On Scroll for scroll-based effects
• Google Fonts and web-safe fonts for typography
• Visual Studio Code as the primary editor

Features

• Animated gradient header for visual impact
• Canvas bubble animation for subtle background motion
• Professional resume layout with clean sections and accent borders
• Responsive design optimized for desktop, tablet, and mobile
• Consistent branding across pages and projects

Sections Included

• Hero section with animated name and summary
• Summary of Qualifications
• Education and Certifications
• Technical Skills
• Work Experience
• Footer with navigation and copyright

Assets
